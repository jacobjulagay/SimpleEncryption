
Compile Instruction on Command Line:
======================================
javac PPMImageDriver.java PPMImage.java
or 
javac *.java


Execution Instruction on Command Line:
======================================
java CS4551_Main [inputfile]
e.g.

java CS4551_Main cat.ppm


Notes:
- Make sure the photo is inside the same folder. 
- .ppm file has to have a comment in the header. You can create a comment by saving a ppm img using infrawview program. Just open the image, then save as and it will automatically create the comment for you.
